金銭管理アプリ

金銭管理アプリ

お金の貸し借りを管理できるWebアプリケーションです。

ログイン機能を搭載しているため、PCでもスマホでもデータを共有して使用できます。

金額や仮払いした人などの情報を入力することで、自動でお金のやり取りを計算してくれます。

プレビュー

使い方

新規登録

1. ユーザ名とパスワードの入力

ユーザ名とパスワードを入力します。
2. 新規登録

Registerボタンをクリックします。
3. ログインページへ

新規登録に成功すると5秒後に自動でログインページに移ります。

ログイン

1. ユーザ名とパスワードの入力

ユーザ名とパスワードを入力します。
2. ログイン

Loginボタンをクリックします。
3. ログイン成功

ログインに成功するとホーム画面に移ります。

ユーザ名変更

1. メニューを開く

右上のメニューボタンをクリックします。
2. メニューの選択

メニューリストから[ユーザ名変更]をクリックします。
3. ユーザ名変更ダイアログ

ユーザ名変更ダイアログが表示されるので、変更後のユーザ名を入力します。

※ユーザ名変更ダイアログが表示されたときに入力欄に入力されているのは現在のユーザ名です。
4. ユーザ名変更

変更ボタンをクリックします。
5. ユーザ名変更完了

ユーザ名の変更は完了です。

パスワード変更

1. メニュー

右上のメニューボタンをクリックします。
2. メニューの選択

メニューリストから[パスワード変更]をクリックします。
3. パスワード変更ダイアログ

パスワード変更ダイアログが表示されるので、変更後のパスワードを入力します。
4. パスワード変更

変更ボタンをクリックします。
5. パスワード変更完了

パスワードの変更は完了です。

プロジェクト追加

1. プロジェクトの追加

画面下中央の＋ボタンをクリックします。
2. プロジェクトの追加完了

新規プロジェクトが追加されます。

プロジェクト並び替え

1. 編集ボタン

画面右下の編集ボタンをクリックします。
2. 編集可能状態

編集可能状態になります。
3. 並び替え

並び替えたいプロンプトをドラッグ&ドロップで移動させます。
4. 並び替え後

ドロップした位置にプロジェクトが移動します。
5. 通常状態

並び替えが終わったら画面右下の編集ボタンをクリックして通常状態に戻します。
6. 並び替え完了

プロジェクトの並び替えは完了です。

プロジェクト削除

1. 編集ボタン

画面右下の編集ボタンをクリックします。
2. 編集可能状態

編集可能状態になります。
3. プロジェクトの削除

削除したいプロジェクトの右上にある×ボタンをクリックします。
4. 削除前確認

確認ダイアログが表示されるので、削除ボタンをクリックします。
5. プロジェクト削除完了

プロジェクトの削除は完了です。

プロジェクトを開く

1. プロジェクトを開く

開きたいプロジェクトをクリックします。
2. 完了

プロジェクトが開きます。

基本操作

1. アイコン変更

アイコンをクリックすることで、アイコンを変更できます。
2. 内容変更

内容セルをクリックし、文字を入力することで内容を変更できます。
3. 金額変更

金額セルをクリックし、文字を入力することで金額を変更できます。
4. 支払者変更

支払者セルをクリックすることで、支払者を変更できます。
5. 支払対象の変更

支払者列より右の列は支払い対象選択列になります。

[○]が支払い対象であることを示し、[-]は支払対象でないことを示します。

[○]と[-]はクリックすることで切り替わります。

行/列の追加

1. 列の追加

列追加ボタンをクリックします。
2. 列の追加完了

列が追加されます。
3. 行の追加

行追加ボタンをクリックします。
4. 行の追加完了

行が追加されます。

行/列の並び替え

1. 列の並び替え①

移動したい列のヘッダを右クリックします。

※スマホの場合は1秒以上長押しします。
2. 列の並び替え②

操作リストが表示されるので、[移動]ボタンをクリックします。
3. 列の並び替え③

列セレクタが表示されるので、移動したい場所のセレクタをクリックします。
4. 列の並び替え完了

列の並び替えは完了です。
5. 行の並び替え①

移動したい行のアイコンを右クリックします。

※スマホの場合は1秒以上長押しします。
6. 行の並び替え②

操作リストが表示されるので、[移動]ボタンをクリックします。
7. 行の並び替え③

行セレクタが表示されるので、移動したい場所のセレクタをクリックします。
8. チャット送信

行の並び替えは完了です。

行/列の削除

1. 列の削除①

移動したい列のヘッダを右クリックします。

※スマホの場合は1秒以上長押しします。
2. 列の削除②

操作リストが表示されるので、[削除]ボタンをクリックします。
3. 列の削除完了

列の削除は完了です。
4. 行の削除①

移動したい行のアイコンを右クリックします。

※スマホの場合は1秒以上長押しします。
5. 行の削除②

操作リストが表示されるので、[削除]ボタンをクリックします。
6. 行の削除完了

行の削除は完了です。

ヘッダ編集

1. メニューを開く

右上のメニューボタンをクリックします。
2. メニューの選択

メニューリストから[ヘッダ編集]をクリックします。
3. ヘッダ編集モード

ヘッダ編集モードになります。

※このモードではヘッダ以外の操作はできません。
4. アイコン変更①

左上のアイコンをクリックします。
5. アイコン変更②

アイコン選択リストが表示されるので、好きなアイコンをクリックします。
6. アイコンの変更完了

アイコンが変更されます。
7. プロジェクト名変更

プロジェクト名をクリックすると入力できる状態になるので、変更後のプロジェクト名を入力します。
8. プロジェクト名の変更完了

プロジェクト名が変更されます。
9. 色の変更①①

ヘッダの右側をクリックします。
10. 色の変更②

カラーセレクタが表示されるので好きな色を選択します。
11. 色の変更完了

ヘッダの色が変更されます。
12. ヘッダ編集の終了

ヘッダの編集を終了するのは、右上のチェックボタンをクリックします。
13. 通常状態

以上でヘッダの編集は終了です。

印刷表示

1. メニューを開く

右上のメニューボタンをクリックします。
2. メニューの選択

メニューリストから[印刷表示]をクリックします。
3. 印刷表示

印刷に適した表示に変更されます。

この状態では、印刷する際に表のデータが欠損することなく全て表示されます。

また、Ctrl+Aで全選択することにより表のデータを全てコピーすることができます。
4. 編集表示

メニューを開き、[編集表示]ボタンをクリックすることで、通常の状態に戻ります。

プロジェクト保存

1. メニューを開く

右上のメニューボタンをクリックします。
2. メニューの選択

メニューリストから[保存]をクリックします。
3. プロジェクトの保存完了

プロジェクトが保存されます。

ログアウト

1. メニューを開く

右上のメニューボタンをクリックします。
2. メニューの選択

メニューリストから[ログアウト]をクリックします。
3. ログアウト完了

ログアウトするとログインページに飛ばされます。
