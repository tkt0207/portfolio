読書アプリ

読書アプリ

画像を読み込んで本形式で閲覧できるWindowsアプリケーションです。

ページをめくる方向や本の開き方を変更できます。

前回のページ位置を記録する機能があるため、途中で閉じても問題ありません。

自動再生を使えば、指定の秒数で自動でページをめくることができます。

プレビュー

使い方

本を開く

1. 本を開く

本の表示をクリックすると本が開きます。
2. 本の選択

本のタイトル文が表示されているブロックをクリックすると、同フォルダ内にある本を選択できます。
3. 選択画面に戻る

ホームボタンをクリックすると本選択画面に戻ります。
4. フォルダを開く

フォルダアイコンをクリックするとフォルダが開きます。
5. 前へ戻る

画面左上の戻るボタンをクリックすると前の画面に戻ります。

めくる方向の変更

1. 出力先選択

ページ方向アイコンをクリックすると、方向選択リストが表示されます
2. 右方向時の操作

画面右側をクリックで次のページへ、左側をクリックで前のページに移動します。

LeftキーとRightキーでも同様の操作が可能です。
3. 左方向時の操作

画面左側をクリックで次のページへ、右側をクリックで前のページに移動します。

LeftキーとRightキーでも同様の操作が可能です。
4. 下方向時の操作

下スクロールで次のページへ、上スクロールで前のページに移動します。

UpキーとDownキーでも同様の操作が可能です。
5. ヘッダの表示/非表示

どのページ方向でも共通で、下にスクロールするとヘッダが非表示になります。

ヘッダを表示したい時には、上にスクロールすることで表示されます。

表示方法の変更

1. 表示方法の変更

表示方法アイコンをクリックすると、表示方法選択リストが表示されます。
2. 片開き

画面に1ページだけ表示します。
3. 両開き(表紙を除外)

表紙は片開きで、それ以降のページは左右両開きに表示します。
4. 両開き(表紙を含む)

全ページを左右両開きで表示します。

自動再生

1. 自動再生の開始

再生ボタンをクリックすると所定の時間でページが自動でめくられるようになります。
2. 自動再生の停止

停止ボタンをクリックするとページの自動再生を止めることができます。
3. 再生時間の設定

再生ボタン(停止ボタン)を右クリックすることで再生時間を設定できます。

再生時間には何msでページを進めるかを指定します。

ページ遷移

1. ページリスト

画面右上のページ数をクリックするとページリストが開きます。
2. ページ選択

リストからページを選択すると、そのページへ移動します。
3. ページ一覧表示

画面左上の一覧アイコンをクリックするとページが一覧で表示されます。
4. ページ一覧からのページ選択

ページをクリックするとそのページへ移動します。

本の読み込み

1. 本の読み込み

画面右上の読み込みボタンをクリックします。
2. 本の選択

選択ウィンドウが表示されるので、読み込みたい本を選択します。
3. 読み込める本の形式

読み込める本の形式は下記のとおりです。
