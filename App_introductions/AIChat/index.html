アプリ紹介

AIと会話ができるWindowsデスクトップアプリケーションです。

オフラインで動作するため、外部に情報が漏れることはありません。

また、AIはプロンプトを入力することで自分好みにカスタマイズが可能です。

ファイルの送信も可能なため、メモ帳のように利用することもできます。

使い方

基本操作

1. チャット送信

入力部に文字を入力し、送信ボタンを押すことでチャットを送信できます。
2. AIからの返答

少し待つとAIからの返答が返ってきます。

ユーザ追加

1. ユーザ追加ボタン

メニューバーの[ユーザ]をクリックし、ユーザ追加ボタンをクリックします。
2. ユーザ追加ウィンドウ

ユーザ追加ウィンドウが表示されます。
3. アイコン設定

アイコン選択部をクリックし、アイコン画像を選択します。
4. ユーザ名設定

ユーザ名入力部にユーザ名を入力します。
5. プロンプト設定

プロンプト入力部にAIへの指示を入力します。
6. ユーザ追加

追加ボタンをクリックします。
7. ユーザ追加完了

ユーザ追加されます。

ユーザ情報変更

1. ユーザ変更ボタン

変更したいユーザを右クリックして、ユーザ変更ボタンをクリックします。
2. ユーザ変更ウィンドウ

ユーザ変更ウィンドウが表示されます。
3. ユーザ情報変更

変更したい箇所を変更し、変更ボタンをクリックします。
4. ユーザ情報変更完了

ユーザ情報が変更されます。

ユーザ削除

1. ユーザ削除ボタン

変更したいユーザを右クリックして、ユーザ削除ボタンをクリックします。
2. ユーザ削除ウィンドウ

ユーザ削除ウィンドウが表示されます。
3. ユーザ削除

OKボタンをクリックすると、ユーザが削除されます。

デザイン設定

1. ユーザリストのデザイン変更

メニューバーの[デザイン]をクリックし、ユーザリストボタンにカーソルを合わせます。
2. ユーザリストのデザイン選択

デザインリストから好きなデザインを選択し、クリックします。
3. ユーザリストのデザイン変更完了

ユーザリストのデザインが変更されます。
4. チャット画面のデザイン変更

メニューバーの[デザイン]をクリックし、チャット画面ボタンにカーソルを合わせます。
5. チャット画面のデザイン選択

デザインリストから好きなデザインを選択し、クリックします。
6. チャット画面のデザイン変更完了

チャット画面のデザインが変更されます。

フォント設定

1. フォントボタン

メニューバーの[デザイン]をクリックし、フォントボタンにカーソルを合わせます。
2. フォント選択

フォントリストから好きなフォントを選択します。
3. フォント変更完了

文字フォントが変更されます。

ファイル送信

1. ファイル送信ボタン

ファイル送信ボタンをクリックします。
2. ファイル選択

好きなファイルを選択し、送信ボタンをクリックします。
3. ファイル送信完了

ファイルが送信されます。

※ファイルの送信ではAIからの返答はありません。

メディア再生

1. 音楽再生

再生ボタンをクリックすることで、音楽を再生できます。
2. 音楽の再生位置変更

スライダーバーをクリックまたは、スライドすることで再生位置を変更できます。
3. 動画再生

再生ボタンをクリックすることで、動画を再生できます。
4. 動画のフルスクリーン化

フルスクリーンボタンをクリックすることで、動画をフルスクリーンで視聴できます。
5. その他

再生速度の変更やピクチャーインピクチャモードにすることもできます。

その他

1. ユーザリストの開閉

ユーザリスト開閉ボタンをクリックすることで、ユーザリストを閉じることができます。
2. ユーザリストが閉じた状態

ユーザリストを閉じることでチャット画面を大きく表示できます。
3. 表示の切り替え

メニューバーの[表示]をクリックすることで、表示の切り替えが可能です。
4. ファイルの保存

送信されたファイルを右クリックするとで、ファイルを保存することができます。
